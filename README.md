# Welcome 👋
- This is a Rest API made With Nodejs, MySQL and deployed on Railway

Link: https://krlospk-rest-api.up.railway.app/api/empleados

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## Run tests

```sh
npm run dev
```
